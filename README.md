# Get Started with Web Development

Welcome to **Get Started with Web Development**—a beginner-friendly live online workshop led by **Jeffrey T. Fritz** on **May 22, 2026**.

No prior web development experience required! In this workshop, you'll learn modern web technologies by building a real app: a **Collection App** where users photograph items and upload photos.

## What You'll Learn

- **Web fundamentals**: HTML structure, semantic markup, and modern CSS
- **Interactive components**: Blazor components, data binding, and event handling
- **Backend skills**: C# basics, data models, methods, and business logic
- **Database persistence**: Entity Framework Core and SQLite
- **Version control**: Git workflows and GitHub collaboration
- **Advanced features**: Photo capture, file uploads, and background processing
- **App orchestration**: Service communication with NServiceBus
- **Observability**: Application monitoring with .NET Aspire
- **AI integration**: Adding AI-powered features with Azure OpenAI
- **Cloud deployment**: Shipping your app to Azure
